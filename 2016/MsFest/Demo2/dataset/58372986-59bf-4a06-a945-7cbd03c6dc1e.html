El primer truco que les mostramos es como hacer que un globo no explote aun cuando este se expone al calor, en este caso cuando se expone a una vela. Sabemos lo que pasa normalmente cuando ponemos un globo sobre una candela y es que este revienta, pero la pregunta es por que cuando el globo tiene agua, este no se revienta.

Bueno es algo muy simple. Al tener agua el globo, esta agua atrae todo el calor y para que este lograra explotar, el agua debería de empezar a calentarse a cierta temperatura que si haría que el globo explote.

Otro truco que resulta ser muy interesante es ver como puede pasar el agua helada a hielo con tan solo un simple golpe. Seguramente muchos han tratado de hacer esto en casa y aunque no siempre lo logramos esto es todo un reto. Lo que sucede en este caso es que con el leve golpe este genera una pequeña cantidad de energía que hace que las moléculas del agua se unan y es ahí donde se forma el hielo.

Y para finalizar con estos trucos estupendos, tenemos como un pedazo de hielo, al ponerle agua helada encima, este sigue creciendo, o sea no se derrite sino que se hace mas grande.
